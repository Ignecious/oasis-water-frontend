# Oasis Water Zimbabwe — Next.js E-Commerce Frontend

> Fresh Next.js 14 implementation of the Oasis Water Zimbabwe e-commerce platform, migrated from Angular.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **State Management:** React Context
- **PDF Generation:** jspdf + jspdf-autotable

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- Product catalog with category filtering (Water, Ice, Dispensers, Accessories)
- Smart cart with MOQ (Minimum Order Quantity) validation
- Multi-step checkout with collection/delivery options
- Zimbabwe payment methods (EcoCash, ZiPIT, Card, Cash)
- Order tracking by order number
- Admin dashboard with product/order management
- PDF invoice generation
- Responsive design (mobile-first)
